SciFinder Web版のユーザー登録

1. 次のリンクをクリックすると、別ブラウザに登録サイトのページが表示されます。
ユーザー登録サイトへ

2. "Click Next to begin registration as a new user." の下の [Next>>] ボタンをクリック。

●注意事項●
　同じPCで続けてユーザー登録を行うと、次のようなメッセージがでて 3 の登録画面が表示されない場合があります。

Suspended URL

User registration is no longer available with this URL.
Please contact the key contact for your organization.
　これは、最初にユーザー登録した情報がブラウザにキャッシュされているために発生するとのことです。
　この場合には、ブラウザの一時保存ファイルを削除してから、再度上の手順 1 から操作をやり直してください。
　（Ie7であれば、[インターネットオプション] → [全般] → [閲覧の履歴] → [削除]。）

3. 個人情報を入力する画面が表示されます。* のついた項目は必須入力の項目です。
項目例（注意事項）
First Name*Taro
Last Name*Ridai
Email*taro@xxx.ous.ac.jp (ous.ac.jp以外のアドレスは登録できません。)
Confirm Email*(確認のためもう一度e-mailアドレスを入力します)
Username*tarord (※下の制限があります)
Password*R&d0it@r- (※下の制限があります)
Re-enter Password*(確認のためもう一度パスワードを入力します)
Security Question*(選択肢より質問を選びます)
Answer*(上の質問への答えを入力します)

※ Username, Password の制限事項
項目制限事項
ユーザー名 ・5〜15文字まで。
・使用できるのは、英字のみ、あるいは英字と数字および次の特殊文字の組み合わせ
　　　（-[ハイフン] _[下線] .[ピリオド] @[アットマーク])
パスワード ・7〜15文字。
・少なくとも次のうち3種類を使用している必要がある。
　　1. 英字
　　2. 大小文字の混合
　　3. 数字
　　4. 英数字以外の文字（例：@, #, %, &, * など）

　以上を入力したら [Registration>>] ボタンをクリック。

4. 初期段階の登録が終了し、CASからe-mailが届くことを示すメッセージが表示されます。

Almost Finished

Thank you for completing the initial step registering to
use SciFinder! You will receive an e-mail message form
CAS with instructions for the registration process.
　この段階ではユーザー登録が終了しただけで、まだ実際にはSciFinder Web版が使える状態にはなっていません。（そのためFinished の前に Almost がついているのです。）

5. CAS より届いたe-mailを開くと、使用上の遵守事項が書かれており、それに同意することを求められています。
　　メールに記載されているリンクをクリックすると、同意したことがCASに登録されます。

6. 登録終了のメッセージが表示されます。
　　これで、3 で取得したUsername, Password でSciFinder Web版に Sign In できるようになります。

Registration for SciFInder is Complete

You have successfully completed the registration process.
To Sign in to SciFinder, click the link below.

https://scifinder.cas.org/scifinder/
